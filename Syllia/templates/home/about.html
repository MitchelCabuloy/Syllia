Syllia is a web-based Syllabus generator made for the professors of De La Salle University. It aims to decrease the time and effort spent in creating and handling Syllabuses. Syllia provides professors with an email like dashboard in where they could manage both Syllabuses and Rubrics, and a simple step by step form in where they could input required information.

Syllia was created by the NetCentric Web Team.

The following was utilized in developing Syllia:

  • Django - Python Web framework that encourages rapid development and clean, pragmatic design.
  • Zurb-Foundation - The most advanced responsive front-end framework in the world.
  • Font Awesome - a full suite of 370 pictographic icons for easy scalable vector graphics on websites.
  • wkhtmltopdf - Simple shell utility to convert html to pdf using the webkit rendering engine, and qt.
